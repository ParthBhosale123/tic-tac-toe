# Tic Tac Toe Game 🎮

A simple and responsive Tic Tac Toe game built with **HTML**, **CSS**, and **JavaScript**. This project uses **Flexbox** for layout and adapts well to different screen sizes.

## 🔥 Features
- Responsive design with Flexbox
- Two-player mode (Player X & Player O)
- Winner detection and display message 🎉
- Draw detection and message if no winner 🤝
- **Restart** button to clear the board
- **New Game** button to reset the whole game
- Smooth user interaction

## 💻 Tech Stack
- HTML5
- CSS3 (Flexbox)
- JavaScript (Vanilla)

## 📱 Responsive Design
The game layout adjusts nicely for mobile, tablet, and desktop screens.
